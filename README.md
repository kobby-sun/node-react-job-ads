# node-react-job-ads
Job Ads checkout system

Steps to run the app
1. install nodejs
2. go to the root folder
3. open terminal or bash, run npm i, wait until its complete
4. run npm run watch
5. open the browser manually and enter http://localhost:8000 to use the app
